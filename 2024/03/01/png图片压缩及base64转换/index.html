python opencv

安装依赖

pip install opencv-python

基本操作

print(img.shape)

输出(448, 504, 3)代表图片就是一个矩阵，每一个像素点都是矩阵中的一个数值而已，对于彩色图片，是采用BGR三通道显示的，也就是一个彩色图片是由三个矩阵组成的，所以上述输出的(448, 504, 3)中，表示是414×500的彩色图片。
